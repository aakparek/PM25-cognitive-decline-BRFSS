# PM25-cognitive-decline-BRFSS
R code for: Associations between PM2.5 and self-reported cognitive decline in the United States, 2015-2023
"This repository contains R code for the analysis of associations between PM2.5 and subjective cognitive decline using BRFSS and EPA data from 2015-2023. Data sources are publicly available from CDC BRFSS and EPA AQS."
